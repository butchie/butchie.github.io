Predicting Biceps Curl Form Based on Accelerometer Data

butchie

September 23, 2015

Executive Summary

A predictive model was constructed to determine what type of biceps curl form, to be called curl class in this analysis, was being used by six individuals asked to perform five different curling motions. By sacrificing generality, a high-performing model was constructed for each individual based on a training dataset. Through the use of classification trees and boosting, accuracies ranging from 91.3% to 99.4% were obtained on validation dataset. Additionally, prediction 100% accuracy was achieved by the models on a small set of test data.

Exploratory Data Analysis

The dataset consisted of the accelerometer measurements attached to six subjects while performing five different biceps curl motions, categorized by the letters ‘A’ through ‘E.’ A total of 19622 observations were made among the 30 combinations of subject and curling motion, with 153 accelerometer measurements listed for each observation. Measurements containing NAs blank values were then removed along with six columns that did not consist of accelerometer data. After transformation, the data consisted of the subject names as well as 53 accelerometer measurements.

To get a better understanding of the data, a scatterplot of two accelerometer measurements, pitch_belt and roll_belt, was constructed as shown in Figure 1. The measurements for each type of curl performed all appear in three different clusters. A second set of scatterplots, one for each test subject, was made and shown in Figure 2. These plots show each individual’s curling data for the same accelerometers on the same scale as the first figure. Based on the relative tightness of each subject’s data, it is apparent that individual differences in form are the cause of the three clusters in the first and not an unknown global cause.

Analysis Methods

The observation made in the previous section allowed for simplification of the analysis. Because the measurements appeared to be affected more by the test subject’s form rather than the type of curl being performed, it was possible to simplify the problem by constructing predictive models for each individual. The simpler could be constructed substantially faster than an overarching one. Even with this simplification, more computationally demanding techniques such as random forests or boosting required further simplification in the dataset due to there being 53 accelerometer measurements for each of several thousand observations for each subject.

Each subject’s data was randomly partitioned by curl class into a training and validation set with a 60/40 ratio. Classification trees were then formed on the training data. The accuracy of the trees was relatively low, ranging from 64.5% for Pedro to 78.3% for Eurico. While this method alone could not be used as a good model, a list of the most important accelerometer measurements with respect to prediction was generated for each subject’s data. The length of this list ranged from 6 for Pedro to 14 for Eurico. More sophisticated techniques were then employed only on these measurements. Boosting using the default parameters through the caret and gbm packages of R was performed on the reduced datasets to try to improve the results of each model.

Cross Validation Results

The results of each subject’s model are shown below. This consists of the model accuracy on the training and validation sets, along with a plot of the cross-validation accuracy for each subject, shown in Figure 3. The models all achieved a training set accuracy of 100% with the exception of Pedro’s, which was 97.6%. While each of these numbers is very high, it is possible that the boosting algorithm overfit the data, resulting in models that are not applicable outside the training data. Cross-validation was performed by predicting curl class on the validation datasets to determine model performance and see whether overfitting was a problem. Accuracies of 97.9% or better were obtained for each subject with the exception of Pedro, where 91.3% of the curling classes were predicted correctly. A qualitative comparison of the accuracy and the spread of data in Figure 2 indicates the individuals with the three largest spreads of data also had the most accurate results. This comparison is limited, however, because more measurements were included for each individual’s predictive model. In all likelihood, Carlitos’ and Jeremy’s data was spread in other variables to allow for good prediction of curl class, while Pedro’s remained the most clustered for all variables, resulting in a lower ability to predict curl class.

Lastly, the models were used to predict the curling class on a small test set of data, consisting of 20 observations. 100% prediction accuracy was obtained, where the small sample size aided the accuracy in this case.

Conclusions

The process of individualizing models for curl class prediction and making use of classification trees and boosting resulted in high cross-validated prediction accuracies ranging for 91.3% to 99.4%. It should be noted that such individualization could only be performed under certain conditions and would not generalize well. The test set included information about which subject was performing the curl to be measured. Without this knowledge, such models could not be used and if attmepted, would not be accurate. Additionally, such models could not generalize to prediction of curling class for any individual. Instead, an overarching model would be needed and would likely require data from a larger population of individuals.
